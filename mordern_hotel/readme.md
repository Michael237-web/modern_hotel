# 🏨 Modern Hotel - Hotel Management Website

A modern, fully responsive hotel management website built with PHP and MySQL. This system allows customers to browse rooms, make bookings, and manage reservations online.

## ✨ Features

### 🌐 Frontend Features
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Room Search & Filter** - Filter rooms by type, price, and availability
- **Online Booking** - Real-time booking with availability check
- **Payment Gateway** - Secure online payment integration
- **Customer Reviews** - Rate and review hotel services
- **Contact Form** - Easy communication with hotel management
- **Image Gallery** - Showcase hotel amenities and rooms
- **Blog Section** - Hotel news and updates

### 🔧 Admin Features
- **Dashboard** - Overview of bookings, revenue, and occupancy
- **Room Management** - Add, edit, delete rooms
- **Booking Management** - View and manage all reservations
- **Customer Management** - Manage customer profiles
- **Payment Management** - Track payments and invoices
- **Reports** - Generate booking and revenue reports
- **User Management** - Manage admin and staff accounts
- **Settings** - Configure website settings

### 💻 Technical Features
- **Secure Authentication** - Admin and user login with password hashing
- **Session Management** - Secure session handling
- **CSRF Protection** - Form security against attacks
- **XSS Prevention** - Input sanitization and validation
- **SQL Injection Prevention** - Prepared statements
- **File Upload** - Image management for rooms
- **Email Notifications** - Booking confirmations and alerts

## 🛠️ Technologies Used

### Backend
- **PHP 7.4+** - Server-side scripting
- **MySQL** - Database management
- **PDO** - Database abstraction layer

### Frontend
- **HTML5** - Markup language
- **CSS3** - Styling and animations
- **JavaScript** - Interactive elements
- **jQuery** - JavaScript library
- **Font Awesome** - Icons
- **Google Fonts** - Typography

### Additional Tools
- **XAMPP/WAMP** - Local development environment
- **Git** - Version control
- **GitHub** - Repository hosting

---

## 📋 Requirements

### Server Requirements
- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache/Nginx web server
- PHP extensions:
  - PDO
  - PDO_MySQL
  - GD (for image processing)
  - mbstring
  - session

### Development Requirements
- XAMPP/WAMP/LAMP stack
- Web browser (Chrome, Firefox, Safari)
- Code editor (VS Code, Sublime Text)
